# 周易64卦 · PWA手机版

## 使用方法

### 方法一：手机浏览器安装（推荐）

1. **把整个 `PWA手机版` 文件夹放到任意静态服务器上**
   - 可以用手机上的 HTTP 服务器 App
   - 也可以部署到 GitHub Pages / Vercel / Netlify 等免费托管
   - 也可以用电脑开个本地服务器，手机同 WiFi 访问

2. **用手机浏览器（Chrome/Safari）打开 index.html**

3. **添加到主屏幕**
   - **安卓 Chrome**：浏览器菜单 → "添加到主屏幕" / "安装应用"
   - **iOS Safari**：分享按钮 → "添加到主屏幕"

4. **从主屏幕图标打开**，就是全屏无浏览器栏的 App 体验，支持离线使用

### 方法二：电脑本地预览

```
# 进入 PWA手机版 目录
cd 周易64卦-打包/PWA手机版

# 用 Node.js 起一个静态服务器
node -e "require('http').createServer((q,s)=>{const f=require('fs'),p=require('path');let fp=q.url==='/'?'index.html':q.url.slice(1);f.readFile(p.join('.',fp),(e,d)=>{if(e){s.writeHead(404);s.end()}else{s.writeHead(200,{'Content-Type':'text/html'});s.end(d)}})}).listen(8080,()=>console.log('http://localhost:8080'))"
```

然后用 Chrome 打开 `http://localhost:8080`，地址栏会出现"安装"按钮，点击即安装为桌面应用。

## 文件说明

| 文件 | 作用 |
|------|------|
| `index.html` | 应用主体（含全部样式和逻辑） |
| `manifest.json` | PWA 配置（名称、图标、主题色） |
| `sw.js` | Service Worker（离线缓存） |
| `icon-192.png` | 应用图标 192×192 |
| `icon-512.png` | 应用图标 512×512 |

## 特性

- ✅ 离线可用（首次加载后缓存）
- ✅ 全屏 App 体验（无浏览器栏）
- ✅ 主屏幕图标
- ✅ 适配手机竖屏
- ✅ 启动速度快（单文件，无外部依赖）
