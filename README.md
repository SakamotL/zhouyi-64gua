# 周易六十四卦 · PWA

学卦（认卦/学卦/用卦）+ 测卦（关键词计算卦象）的手机端 PWA 应用。

## 在线访问

部署到 GitHub Pages 后，访问：`https://你的用户名.github.io/仓库名/`

## 手机安装

1. 手机浏览器打开上面网址
2. 安卓 Chrome：菜单 → "添加到主屏幕"
3. iOS Safari：分享 → "添加到主屏幕"
4. 从主屏幕图标打开，全屏 App 体验，支持离线

## 功能

- **认卦**：阴阳爻、八卦、64卦矩阵、爻位命名
- **学卦**：吉卦/凶卦/转机卦/关系卦/行事卦 5类主题
- **用卦**：占卜方法、解卦三步法、占卜模拟器、实战案例
- **测卦**：输入关键词，本地公式计算匹配卦象，含计算过程可视化
